# 🔥 1. Asosiy konsept (Product mindset)

Platforma 3 ta asosiy rolga ega:

1. **Oddiy user**

   * kitob joylaydi (sotish / bepul / vaqtincha)
   * boshqa e’lonlarni ko‘radi
   * xaritada yaqinlarini topadi

2. **Kutubxona / Do‘kon egasi**

   * o‘z “store” profilini yaratadi
   * kitoblar katalogini joylaydi
   * lokatsiya majburiy

3. **Admin (keyin qo‘shiladi)**

   * spam nazorat
   * userlarni bloklash
   * e’lonlarni tekshirish
